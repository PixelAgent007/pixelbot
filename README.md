## Requirements
+ Python => 3.7.3
+ Diverse pip packages (install by running):

`python -m pip install -r requirements.txt`

(or `python3 -m pip install -r requirements.txt`, depends on your setup)

+ ffmpeg
+ A PostgreSQL server
